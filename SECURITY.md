\# Security Policy



Este repositório documenta meu portfólio técnico e laboratórios de estudo em Redes, Infraestrutura, Cloud Computing e Segurança da Informação.



\## Dados que não devem ser publicados



Antes de publicar qualquer imagem, print, log, comando, configuração ou evidência, devo verificar se não existe exposição de:



\- Senhas

\- Tokens

\- Chaves privadas

\- IP público

\- Credenciais de dashboards

\- Cookies ou sessões

\- E-mails pessoais em prints

\- MAC addresses reais

\- Hostnames pessoais

\- Caminhos locais sensíveis

\- Arquivos `.env`

\- Dados de terceiros



\## Evidências



Prints, logs e imagens técnicas podem conter informações sensíveis. Quando necessário, os dados devem ser anonimizados antes da publicação.



\## Padrão de anonimização



| Tipo de dado | Padrão recomendado |

|---|---|

| IP público | `x.x.x.x` |

| Usuário | `usuario-lab` |

| Senha | `\*\*\*\*\*\*\*\*` |

| Token | `<TOKEN\_REMOVIDO>` |

| E-mail | `email@example.com` |

| MAC address | `00:00:00:00:00:00` |

| Hostname | `host-lab` |

| Domínio interno | `lab.local` |



\## Contato



Caso encontre algum dado sensível neste repositório, entre em contato pelo LinkedIn ou pelo e-mail profissional informado no perfil do GitHub.
