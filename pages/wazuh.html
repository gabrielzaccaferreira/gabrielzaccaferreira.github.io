🛡️ CASE TÉCNICO

Wazuh SIEM/XDR

Implementação de um laboratório completo de monitoramento e detecção de ameaças utilizando Ubuntu Server, VMware Workstation e Kali Linux, com foco em SIEM, análise de eventos e segurança da informação.

Ver Arquitetura Ver GitHub

Arquitetura

Ambiente do laboratório

O laboratório foi implementado em ambiente virtualizado utilizando VMware Workstation, com um servidor Ubuntu para os componentes centrais do Wazuh e uma máquina Kali Linux atuando como endpoint monitorado.

🖥️ Host

Dell OptiPlex com Windows 11, utilizado como base para virtualização.

⚙️ Hypervisor

VMware Workstation para criação e gerenciamento das máquinas virtuais.

🐧 Ubuntu Server

Servidor principal executando Wazuh Manager, Dashboard, Indexer e Filebeat.

💻 Kali Linux

Endpoint monitorado com agente Wazuh instalado e integrado ao servidor.

Tecnologias

Stack utilizada no laboratório.

Ubuntu Server VMware Workstation Wazuh Kali Linux Filebeat Linux

Aprendizados

Principais conhecimentos adquiridos.

SIEM

Compreensão do funcionamento de uma plataforma de monitoramento e correlação de eventos.

Desafios

Problemas encontrados e soluções aplicadas.

⚠️ Dashboard

Diagnóstico e validação dos serviços do Wazuh até a disponibilização correta da interface.

⚠️ Comunicação do agente

Configuração do XRDP e ambiente gráfico para administração remota do Kali Linux.

Conclusão

Resultado do laboratório.

Este laboratório consolidou conhecimentos práticos em SIEM, Linux, virtualização, agentes de monitoramento, análise de eventos e documentação técnica. O projeto representa uma etapa importante na construção do meu portfólio voltado para Redes, Infraestrutura e Cibersegurança.
